---

## Method

```php
create(string $paymentId, array $payload = []): mixed
```

## Behavior

- if `amount` exists, the package calls the partial refund operation
- if `amount` is omitted, the package calls the full refund operation

## Examples

```php
$fullRefund = $refundService->create('1234567890');

$partialRefund = $refundService->create('1234567890', [
    'amount' => 25.00,
]);
```
