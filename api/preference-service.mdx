description: API reference for creating Mercado Pago checkout preferences with PreferenceService.
---

## Method

```php
create(array $payload): mixed
```

## Accepted payload keys

- `items` required
- `payer`
- `back_urls`
- `metadata`
- `notification_url`
- `external_reference`
- `expires`
- `expiration_date_from`
- `expiration_date_to`

## Example

```php
$preference = $preferenceService->create([
    'items' => [
        [
            'title' => 'Pro plan',
            'quantity' => 1,
            'unit_price' => 49.90,
        ],
    ],
    'payer' => [
        'email' => 'buyer@example.com',
    ],
]);
```

## Typical output fields

- `id`
- `init_point`
- `sandbox_init_point`
