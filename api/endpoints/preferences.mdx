## Availability

This endpoint only responds when demo routes are enabled and the app environment is `local` or `testing`.

## Route

```text
POST /api/mercadopago/preferences
```

## Required fields

- `items[].title`
- `items[].quantity`
- `items[].unit_price`

## Optional fields

- `payer`
- `back_urls`
- `metadata`
- `notification_url`
- `external_reference`
- `expires`
- `expiration_date_from`
- `expiration_date_to`

## Success response

```json
{
  "ok": true,
  "data": {
    "id": "1234567890",
    "init_point": "https://www.mercadopago.com/checkout/v1/redirect?pref_id=..."
  },
  "meta": []
}
```
