## Availability

This endpoint only responds when demo routes are enabled and the app environment is `local` or `testing`.

## Route

```text
POST /api/mercadopago/test-users
```

## Required payload

- `site_id` must be one of `MLA`, `MLB`, `MLC`, `MLM`, `MLU`, `MCO`, `MPE`

Optional:

- `description`
