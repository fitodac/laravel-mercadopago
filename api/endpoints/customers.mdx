## Availability

These endpoints only respond when demo routes are enabled and the app environment is `local` or `testing`.

## Routes

```text
POST /api/mercadopago/customers
GET /api/mercadopago/customers/{customerId}
POST /api/mercadopago/customers/{customerId}/cards
DELETE /api/mercadopago/customers/{customerId}/cards/{cardId}
```

## Customer create required field

- `email`

## Card create required field

- `token`
