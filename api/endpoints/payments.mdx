## Availability

These endpoints only respond when demo routes are enabled and the app environment is `local` or `testing`.

## Routes

```text
POST /api/mercadopago/payments
GET /api/mercadopago/payments/{paymentId}
POST /api/mercadopago/payments/{paymentId}/refunds
```

## Create payment required fields

- `transaction_amount`
- `token`
- `description`
- `installments`
- `payment_method_id`
- `payer.email`

## Refund payload

`amount` is optional. If omitted, the package requests a full refund.
