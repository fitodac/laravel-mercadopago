---
title: Webhooks endpoint
description: Always-on endpoint reference for Mercado Pago webhook notifications.
---

## Availability

This endpoint is always active in every environment.

## Route

```text
POST /api/mercadopago/webhooks
```

## Accepted request data

- `type`
- `action`
- `api_version`
- `data.id`
- `live_mode`
- `user_id`
- query string `topic`

## Signature validation

When `MERCADOPAGO_WEBHOOK_SECRET` is configured and `x-signature` is present, the package validates:

- `x-signature`
- `x-request-id`
- `data.id`

## Success response

```json
{
  "ok": true,
  "data": {
    "acknowledged": true,
    "validated": true,
    "topic": "payment",
    "resource": "1234567890"
  },
  "meta": []
}
```
