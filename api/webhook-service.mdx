---
title: WebhookService
description: API reference for webhook handling and signature validation in Laravel MercadoPago.
---

## Method

```php
handle(Request $request): array
```

## Returned fields

- `acknowledged`
- `validated`
- `topic`
- `resource`
- `payload`

## Validation rules

Signature validation runs only when:

- `MERCADOPAGO_WEBHOOK_SECRET` is configured
- `x-signature` is present in the request

## Example output

```json
{
  "acknowledged": true,
  "validated": true,
  "topic": "payment",
  "resource": "1234567890",
  "payload": {
    "type": "payment",
    "data": {
      "id": "1234567890"
    }
  }
}
```
