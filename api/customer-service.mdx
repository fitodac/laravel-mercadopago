---
title: CustomerService
description: API reference for creating and retrieving Mercado Pago customers.
---

## Methods

```php
create(array $payload): mixed
get(string $customerId): mixed
```

## Required create fields

- `email`

Optional:

- `first_name`
- `last_name`
- `phone`
- `identification`
- `default_address`
- `metadata`

## Example

```php
$customer = $customerService->create([
    'email' => 'buyer@example.com',
    'first_name' => 'Jane',
    'last_name' => 'Doe',
]);

$sameCustomer = $customerService->get((string) data_get($customer, 'id'));
```
