title: PaymentMethodService
description: API reference for listing available Mercado Pago payment methods.
---

## Method

```php
all(): mixed
```

## Example

```php
$paymentMethods = $paymentMethodService->all();
```

## Notes

This service is read-only. It is useful for frontend payment forms and for validating which methods are available in your Mercado Pago account context.
