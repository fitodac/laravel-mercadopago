## Method

```php
create(array $payload): mixed
```

## Required payload

- `site_id` must be one of `MLA`, `MLB`, `MLC`, `MLM`, `MLU`, `MCO`, `MPE`

Optional:

- `description`

## Example

```php
$testUser = $testUserService->create([
    'site_id' => 'MLA',
    'description' => 'Local QA user',
]);
```
