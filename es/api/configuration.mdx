## Claves de config

| Key | Type | Default |
| --- | --- | --- |
| `mercadopago.access_token` | `string|null` | `null` |
| `mercadopago.public_key` | `string|null` | `null` |
| `mercadopago.webhook_secret` | `string|null` | `null` |
| `mercadopago.route_prefix` | `string` | `api/mercadopago` |
| `mercadopago.enable_demo_routes` | `bool` | `true` |
| `mercadopago.runtime_environment` | `string|null` | `null` |
| `mercadopago.credential_resolver` | `class-string` | `EnvCredentialResolver::class` |

## Nombres de rutas registradas
