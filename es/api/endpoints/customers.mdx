## Disponibilidad

Estos endpoints solo responden cuando las demo routes están habilitadas y el entorno de la app es `local` o `testing`.

## Rutas

```text
POST /api/mercadopago/customers
GET /api/mercadopago/customers/{customerId}
POST /api/mercadopago/customers/{customerId}/cards
DELETE /api/mercadopago/customers/{customerId}/cards/{cardId}
```

## Campo obligatorio para create customer

- `email`

## Campo obligatorio para create card

- `token`
