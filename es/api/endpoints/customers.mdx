## Resumen

La API de Customers aloja aquellas rutas destinadas a crear legajos crudos de clientes en la nube de MP, a recuperar sus datos y a gestionar la amalgama de tarjetas de crédito prepagas que hayan guardado. Útil en flujos de "suscripciones" y "one-click checkout".

<Warning>
  Como norma inquebrantable de Seguridad, todas las URL listadas en este archivo nacen "Inertes" o dan Error 404 en el momento en que Laravel corra bajo modo Producción. 
  Solo resucitan si configurás expresamente `MERCADOPAGO_ENABLE_DEMO_ROUTES=true` y tu base corre en `APP_ENV=local|testing`.
</Warning>

## Creación Limpia de Clientela

Inventar un perfil nuevo.

### Endpoint

```
POST /api/mercadopago/customers
```

## Buscando a un Cliente (`GET`)

Acarrear un viejo cliente para mirar sus adentros usando su identificador encriptado.

### Endpoint

```
GET /api/mercadopago/customers/{customerId}
```

### Patrón Obligatorio

<ParamField path="customerId" type="string" required>
  La estampa del perfil que devolvió la ruta POST de Creación de Clientela.
</ParamField>

## Anestesiando (Atando) una Tarjeta Tokenizada (`POST`)

Amarrar tarjetas a los legajos perpetuamente.

### Endpoint

```
POST /api/mercadopago/customers/{customerId}/cards
```

### Patrones de Ruta y Cuerpo

<ParamField path="customerId" type="string" required>
  ID alfanumérico del Comprador.
</ParamField>

<ParamField path="token" type="string" required>
  La representación encriptada (`El Token de Cartón Seguro`).
</ParamField>

### Respuesta Otorgada

<ResponseField name="data" type="object">
  Tu tarjeta quedó subida a la nube.
  <Expandable title="Datazo Vital Devueltos">
    <ResponseField name="id" type="string">
      El ID propio y minúsculo **para referirse de manera unívoca a ESE plástico**. (A diferencia del CustomerID, sirve para cobrar).
    </ResponseField>

    <ResponseField name="first_six_digits" type="string">
      Descubrimiento primario (Por ej: 450995 de VISA Débito).
    </ResponseField>

    <ResponseField name="last_four_digits" type="string">
      Identidad final de validación (terminada en 0014).
    </ResponseField>
  </Expandable>
</ResponseField>

## Retiro Espontáneo de un Medio de Abono (`DELETE`)

Quitar de la nómina una tarjeta particular de las garras de suscripción en caliente.

### Endpoint

```
DELETE /api/mercadopago/customers/{customerId}/cards/{cardId}
```

### Path Parameters

<ParamField path="customerId" type="string" required>
  El jefe macro.
</ParamField>

<ParamField path="cardId" type="string" required>
  Id hijo en particular de un plástico (Obtenido en la creación o en un GET global).
</ParamField>

## Detalles de Operabilidad

Al salir a jugar fuera del cajón de aserrín de Localhost (`APP_ENV=local`), deberás armar imperativamente un Controlador de Laravel vos mismo bajo un API Resource en tus propios archivos (`routes/api.php`) amparado por los escudos Middleware de Sessión o Tokenización CSRF (`auth:sanctum`), para evitar que intrusos te colapsen el Rate Limit subiendo tarjetitas ajenas.
