description: Referencia del endpoint demo para listar payment methods disponibles.
---

## Disponibilidad

Este endpoint solo responde cuando las demo routes están habilitadas y el entorno de la app es `local` o `testing`.

## Ruta

```text
GET /api/mercadopago/payment-methods
```

## Respuesta

Devuelve la respuesta normalizada del SDK para los payment methods disponibles en el contexto actual de credenciales.
