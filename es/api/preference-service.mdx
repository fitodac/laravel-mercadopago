---
title: PreferenceService
description: Referencia API para crear checkout preferences de Mercado Pago con PreferenceService.
---

## Método

```php
create(array $payload): mixed
```

## Claves de payload aceptadas

- `items` obligatorio
- `payer`
- `back_urls`
- `metadata`
- `notification_url`
- `external_reference`
- `expires`
- `expiration_date_from`
- `expiration_date_to`

## Ejemplo

```php
$preference = $preferenceService->create([
    'items' => [
        [
            'title' => 'Pro plan',
            'quantity' => 1,
            'unit_price' => 49.90,
        ],
    ],
    'payer' => [
        'email' => 'buyer@example.com',
    ],
]);
```

## Campos típicos de salida

- `id`
- `init_point`
- `sandbox_init_point`
