---
title: WebhookService
description: Referencia API para manejo de webhooks y validación de firmas en Laravel MercadoPago.
---

## Método

```php
handle(Request $request): array
```

## Campos devueltos

- `acknowledged`
- `validated`
- `topic`
- `resource`
- `payload`

## Reglas de validación

La validación de firma corre solo cuando:

- `MERCADOPAGO_WEBHOOK_SECRET` está configurado
- `x-signature` está presente en la request

## Ejemplo de salida

```json
{
  "acknowledged": true,
  "validated": true,
  "topic": "payment",
  "resource": "1234567890",
  "payload": {
    "type": "payment",
    "data": {
      "id": "1234567890"
    }
  }
}
```
