---
title: CardService
description: Referencia API para agregar y eliminar customer cards con CardService.
---

## Métodos

```php
create(string $customerId, array $payload): mixed
delete(string $customerId, string $cardId): mixed
```

## Payload obligatorio

- `token`

## Ejemplo

```php
$card = $cardService->create('123456', [
    'token' => 'card-token',
]);

$result = $cardService->delete('123456', '987654');
```
