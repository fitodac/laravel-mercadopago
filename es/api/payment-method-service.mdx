---
title: PaymentMethodService
description: Referencia API para listar payment methods disponibles de Mercado Pago.
---

## Método

```php
all(): mixed
```

## Ejemplo

```php
$paymentMethods = $paymentMethodService->all();
```

## Notas

Este service es solo de lectura. Sirve para formularios frontend de pago y para validar qué métodos están disponibles en el contexto de tu cuenta de Mercado Pago.
