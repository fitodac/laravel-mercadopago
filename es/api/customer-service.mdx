---
title: CustomerService
description: Referencia API para crear y consultar customers de Mercado Pago.
---

## Métodos

```php
create(array $payload): mixed
get(string $customerId): mixed
```

## Campo obligatorio para create

- `email`

Opcionales:

- `first_name`
- `last_name`
- `phone`
- `identification`
- `default_address`
- `metadata`

## Ejemplo

```php
$customer = $customerService->create([
    'email' => 'buyer@example.com',
    'first_name' => 'Jane',
    'last_name' => 'Doe',
]);

$sameCustomer = $customerService->get((string) data_get($customer, 'id'));
```
