---
title: TestUserService
description: Referencia API para crear test users de Mercado Pago con TestUserService.
---

## Método

```php
create(array $payload): mixed
```

## Payload obligatorio

- `site_id` debe ser uno de `MLA`, `MLB`, `MLC`, `MLM`, `MLU`, `MCO`, `MPE`

Opcional:

- `description`

## Ejemplo

```php
$testUser = $testUserService->create([
    'site_id' => 'MLA',
    'description' => 'Local QA user',
]);
```
