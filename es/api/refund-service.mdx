---
title: RefundService
description: Referencia API para full refunds y partial refunds con RefundService.
---

## Método

```php
create(string $paymentId, array $payload = []): mixed
```

## Comportamiento

- si existe `amount`, el paquete llama a la operación de partial refund
- si omitís `amount`, el paquete llama a la operación de full refund

## Ejemplos

```php
$fullRefund = $refundService->create('1234567890');

$partialRefund = $refundService->create('1234567890', [
    'amount' => 25.00,
]);
```
