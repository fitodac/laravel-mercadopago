---
title: PaymentService
description: Referencia API para crear y consultar payments con PaymentService.
---

## Métodos

```php
create(array $payload): mixed
get(string $paymentId): mixed
```

## Campos obligatorios para create

- `transaction_amount`
- `token`
- `description`
- `installments`
- `payment_method_id`
- `payer.email`

## Ejemplo

```php
$payment = $paymentService->create([
    'transaction_amount' => 150.75,
    'token' => 'card-token',
    'description' => 'Order #1001',
    'installments' => 1,
    'payment_method_id' => 'visa',
    'payer' => [
        'email' => 'buyer@example.com',
    ],
]);

$samePayment = $paymentService->get('1234567890');
```
