## Panorama general

Laravel MercadoPago es un paquete reutilizable para proyectos Laravel 12 que necesitan una capa de integración clara con Mercado Pago.

Registra:

- `LaravelMercadoPagoServiceProvider` mediante auto-discovery de Laravel
- `config/mercadopago.php`
- servicios singleton para operaciones de Mercado Pago
- endpoints JSON demo bajo un prefijo configurable
- un endpoint de webhook siempre activo

## Qué resuelve el paquete

- Resuelve credenciales desde configuración mediante `CredentialResolverInterface`
- Configura el access token del SDK de Mercado Pago y su runtime environment
- Envuelve clientes del SDK en servicios inyectables de Laravel
- Normaliza respuestas de servicios en envelopes JSON consistentes para los endpoints demo
- Valida firmas de webhook cuando `MERCADOPAGO_WEBHOOK_SECRET` está configurado

## Servicios incluidos

| Service | Propósito |
| --- | --- |
| `PreferenceService` | Crear checkout preferences |
| `PaymentService` | Crear y consultar payments |
| `RefundService` | Procesar full refunds y partial refunds |
| `CustomerService` | Crear y consultar customers |
| `CardService` | Agregar y eliminar customer cards |
| `PaymentMethodService` | Listar payment methods disponibles |
| `WebhookService` | Procesar payloads de webhook y validar firmas |
| `TestUserService` | Crear test users de Mercado Pago |

## Qué no incluye el paquete

- Migraciones de base de datos
- Componentes UI de checkout
- Persistencia de payments en tu base de datos
- Autenticación o autorización para tus endpoints de negocio
- Queue workers, retries o almacenamiento de webhooks

<Warning>
  Los endpoints demo son herramientas de apoyo. Para flujos de producción,
  inyectá los services en tus propios controllers, jobs o actions y aplicá tus
  reglas de auth, validación y persistencia.
</Warning>

## Requisitos

- PHP `^8.2`
- Componentes Laravel `^12.0`
- `mercadopago/dx-php ^3.8`

## Envelope de respuesta

Los demo controllers del paquete devuelven JSON con esta forma:

```json
{
  "ok": true,
  "data": {},
  "meta": []
}
```

Los errores devuelven:

```json
{
  "ok": false,
  "message": "Mercado Pago access token is not configured."
}
```

Las respuestas de `MPApiException` también pueden incluir `details` con el contenido devuelto por el SDK.
