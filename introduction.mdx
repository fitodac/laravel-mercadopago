## Overview

Laravel MercadoPago is a reusable package for Laravel 12 projects that need a clean integration layer for Mercado Pago.

It registers:

- `LaravelMercadoPagoServiceProvider` through Laravel auto-discovery
- `config/mercadopago.php`
- singleton services for Mercado Pago operations
- JSON demo endpoints under a configurable prefix
- an always-on webhook endpoint

## What the package solves

- Resolves credentials from configuration through `CredentialResolverInterface`
- Configures the Mercado Pago SDK access token and runtime environment
- Wraps SDK clients in injectable Laravel services
- Normalizes service responses into consistent JSON envelopes for demo endpoints
- Validates webhook signatures when `MERCADOPAGO_WEBHOOK_SECRET` is configured

## Included services

| Service | Purpose |
| --- | --- |
| `PreferenceService` | Create checkout preferences |
| `PaymentService` | Create and fetch payments |
| `RefundService` | Process full and partial refunds |
| `CustomerService` | Create and fetch customers |
| `CardService` | Add and delete customer cards |
| `PaymentMethodService` | List available payment methods |
| `WebhookService` | Handle webhook payloads and signature checks |
| `TestUserService` | Create Mercado Pago test users |

## What the package does not include

- Database migrations
- Checkout UI components
- Payment persistence in your app database
- Authentication or authorization for your own business endpoints
- Queue workers, retries, or webhook storage

<Warning>
  Demo endpoints are support tooling. For production flows, inject the services
  into your own controllers, jobs, or actions and apply your own auth,
  validation, and persistence rules.
</Warning>

## Requirements

- PHP `^8.2`
- Laravel components `^12.0`
- `mercadopago/dx-php ^3.8`

## Response envelope

The package demo controllers return JSON in this shape:

```json
{
  "ok": true,
  "data": {},
  "meta": []
}
```

Failures return:

```json
{
  "ok": false,
  "message": "Mercado Pago access token is not configured."
}
```

`MPApiException` responses may also include a `details` field with SDK response content.
