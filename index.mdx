# Laravel MercadoPago

Documentation for the reusable Laravel package at [fitodac/laravel-mercadopago](https://github.com/fitodac/laravel-mercadopago).

The package provides:

- Laravel auto-discovery for `LaravelMercadoPagoServiceProvider`
- Reusable services for preferences, payments, customers, cards, refunds, webhooks, payment methods, and test users
- A configurable JSON demo API for `local` and `testing`
- An always-on webhook endpoint with optional HMAC validation
