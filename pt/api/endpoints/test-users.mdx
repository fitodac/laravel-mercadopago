---

## Disponibilidade

Este endpoint só responde quando as demo routes estão habilitadas e o ambiente da app é `local` ou `testing`.

## Rota

```text
POST /api/mercadopago/test-users
```

## Payload obrigatório

- `site_id` deve ser um de `MLA`, `MLB`, `MLC`, `MLM`, `MLU`, `MCO`, `MPE`

Opcional:

- `description`
