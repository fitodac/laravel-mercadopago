---
title: CardService
description: Referência API para adicionar e remover customer cards com CardService.
---

## Métodos

```php
create(string $customerId, array $payload): mixed
delete(string $customerId, string $cardId): mixed
```

## Payload obrigatório

- `token`

## Exemplo

```php
$card = $cardService->create('123456', [
    'token' => 'card-token',
]);

$result = $cardService->delete('123456', '987654');
```
